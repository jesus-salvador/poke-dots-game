# How to play
this is a basic game just mouse click to play 

# Cheatsheet
```
# create a virtualenv
virtualenv venv
source venv/bin/activate
pip install -r dev-requirements.txt
```

# To run from command-line (Mac, Linux):
```
  python3 poke-v1.py
  python3 poke-v2.py
  python3 poke-v3.py
  python3 poke-v4.py
  python3 poke-v5.py
```
